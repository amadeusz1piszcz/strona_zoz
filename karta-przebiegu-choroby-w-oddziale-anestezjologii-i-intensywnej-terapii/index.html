KARTA PRZEBIEGU CHOROBY W ODDZIALE ANESTEZJOLOGII I INTENSYWNEJ TERAPII

I. Specyfikacja Wymagań Dokumentacyjnych

Dane Administracyjne i Parametry Wyjściowe

Dane identyfikacyjne oraz kluczowe parametry fizjologiczne niezbędne do dawkowania leków i wentylacji.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Identyfikacja Podmiotu

Pełna nazwa szpitala, kod resortowy, nazwa OAIIT.

System / Admin

2. Identyfikacja Pacjenta

Imię, nazwisko, PESEL, nr Księgi Głównej, Wiek.

Rejestracja / Piel.

3. Doba Pobytu

Kolejny numer doby pobytu w OAiIT (krytyczne dla oceny przebiegu sepsy/antybiotykoterapii).

System / Piel.

4. Dane Antropometryczne

Waga (rzeczywista/należna), Wzrost, BMI, Powierzchnia ciała (BSA) – niezbędne do przeliczania dawek (mcg/kg/min) i indeksowania parametrów rzutu serca.

Lekarz / Piel.

5. Rozpoznanie/Problem

Główne rozpoznanie lub powód przyjęcia (np. "Wstrząs septyczny", "Niewydolność oddechowa").

Lekarz

6. Alerty (Izolacja)

Status epidemiologiczny: "Izolacja kontaktowa (CPE/VRE)", "Izolacja oddechowa", "Alergie".

Lekarz / Piel.

Monitorowanie Hemodynamiczne i Ogólne (Karta Godzinowa)

Rejestr parametrów życiowych prowadzony w ścisłym reżimie czasowym (zazwyczaj co 1h).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Znacznik Czasowy

Godzina pomiaru (np. 08:00, 09:00...). W stanach krytycznych częściej (np. co 15 min).

Pielęgniarka

2. Układ Krążenia

Tętno (HR), Ciśnienie tętnicze (NIBP lub IBP - inwazyjne), Ośrodkowe Ciśnienie Żylne (OCŻ/CVP), Saturacja (SpO2).

Pielęgniarka

3. Temperatura

Temperatura ciała (centralna/powierzchniowa) oraz metoda regulacji (np. "Koc grzewczy", "Metoda wewnątrznaczyniowa").

Pielęgniarka

4. Skala GCS

Ocena przytomności w skali Glasgow (E+V+M). Jeśli pacjent zaintubowany/sedowany – adnotacja (np. "S").

Pielęgniarka

5. Źrenice

Ocena szerokości (mm) i reakcji na światło (P/L).

Pielęgniarka

6. Kolor Skóry / Nawrót

Ocena perfuzji obwodowej: nawrót kapilarny (CRT), obecność sinicy, marmurkowatość.

Pielęgniarka

Układ Oddechowy i Wentylacja Mechaniczna

Parametryzacja wsparcia oddechowego (respiratora).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Tryb Wentylacji

Ustawiony tryb (np. SIMV, PCV, PSV, CPAP) lub "Oddech własny/Tlenoterapia bierna".

Lekarz (zlec.) / Piel.

2. Parametry Ustawione

FiO2 (%), PEEP (cmH2O), Częstość oddechów (f), Objętość oddechowa (TV), Ciśnienie szczytowe/wspierające (Pinsp/Psupp).

Pielęgniarka

3. Parametry Mierzone

Rzeczywista objętość minutowa (MV), Ciśnienie szczytowe w drogach oddechowych (Ppeak), EtCO2 (kapnometria).

Pielęgniarka

4. Mechanika Oddychania

(Okresowo) Podatność płuc, Opory  – wpis lekarski.

Lekarz

5. Toaleta Dróg Oddechowych

Odnotowanie odsysania wydzieliny (ilość, charakter: ropna/krwista/śluzowa) oraz higieny jamy ustnej (profilaktyka VAP).

Pielęgniarka

Bilans Płynów

Precyzyjne monitorowanie gospodarki wodno-elektrolitowej (zapis godzinowy i dobowy).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Podaż (Wlewy)

Objętość krystaloidów, koloidów, preparatów krwiopochodnych, żywienia pozajelitowego/dojelitowego oraz leków w pompach (ml/h).

Pielęgniarka

2. Wydalanie (Diureza)

Ilość moczu (ml) mierzona godzinowo (Diureza godzinowa). Jeśli <0.5 ml/kg/h – alert.

Pielęgniarka

3. Inne Straty

Objętość treści z drenów, sondy żołądkowej, biegunka, wymioty, ultrafiltracja (CRRT).

Pielęgniarka

4. Bilans Dobowy

Sumaryczne podliczenie: Podaż minus Wydalanie. Wynik: Dodatni (+) lub Ujemny (-).

Pielęgniarka

Farmakoterapia (Karta Zleceń Intensywnych)

Zarządzanie lekami podawanymi w trybie ciągłym (wlew) i doraźnym.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Aminy Katecholowe

Nazwa leku (Noradrenalina, Adrenalina, Dobutamina). Zlecenie: dawka w mcg/kg/min lub ml/h. Rejestracja zmian przepływu w czasie.

Lekarz (zlec.) / Piel.

2. Analgosedacja

Leki sedacyjne (Propofol, Midazolam) i przeciwbólowe (Fentanyl, Morfina). Dawkowanie ciągłe.

Lekarz (zlec.) / Piel.

3. Antybiotykoterapia

Nazwa, dawka, droga podania. Godzina podania (rygorystyczne przestrzeganie, szczególnie przy TDM - monitorowaniu stężenia).

Pielęgniarka

4. Profilaktyka Przeciwzakrzepowa

Heparyny drobnocząsteczkowe/niefrakcjonowane (dawka, godzina).

Pielęgniarka

Skale Oceny Stanu i Bezpieczeństwa (Monitoring Specjalistyczny)

Narzędzia obiektywizujące stan pacjenta (wymogi CMJ i wytycznych anestezjologicznych).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Ocena Bólu i Sedacji

Skala RASS/SAS (poziom pobudzenia/sedacji) oraz CPOT/BPS (ból u pacjentów niekomunikujących się). Ocena cykliczna (np. co 4h).

Pielęgniarka

2. Ocena Majaczenia (Delirium)

Skala CAM-ICU (wykonywana min. raz na zmianę/dobę). Wynik: Pozytywny/Negatywny.

Pielęgniarka / Lekarz

3. Skale Ciężkości

APACHE II/III, SOFA, TISS-28 (ocena nakładu pracy i rokowania) – zazwyczaj raz na dobę.

Lekarz

4. Kontrola Dostępów

Lista założonych linii (CVC, tętnicza, drenaże). Data założenia i ocena miejsca wkłucia (V.I.P.). Decyzja o utrzymaniu.

Lekarz / Piel.

II. Standardy Akredytacyjne CMJ 2025 (Kluczowe Mechanizmy)

Dokumentacja OAiIT jest najbardziej szczegółowym rejestrem w szpitalu. Kluczowe obszary:

  1. Pakiety Zadaniowe:

    1. Odrespiratorowe Zapalenie Płuc (VAP): Karta musi zawierać pola potwierdzające: uniesienie wezgłowia (30-45°), higienę jamy ustnej chlorheksydyną, kontrolę ciśnienia w mankiecie rurki intubacyjnej, codzienne "okno sedacyjne" (próba wybudzenia).

    2. Zakażenia Odcewnikowe (CLABSI): Codzienna ocena konieczności utrzymania wkłucia centralnego (CVC).
  2. Zarządzanie Bólem, Pobudzeniem i Delirium (PAD): Standardy wymagają stosowania protokołu PAD. Karta musi wymuszać cykliczną ocenę (nie rzadziej niż co 4-8h) w skalach RASS/CPOT/CAM-ICU. Brak oceny bólu u pacjenta nieprzytomnego jest błędem.
  3. Bilans Płynów: Precyzyjny bilans jest podstawą terapii w OAIIT. Dokumentacja musi sumować bilanse cząstkowe (godzinowe) w bilans dobowy i skumulowany, co pozwala na ocenę przewodnienia pacjenta.
  4. Raportowanie Zdarzeń Niepożądanych: Karta powinna umożliwiać szybkie odnotowanie zdarzeń takich jak: ekstubacja (przypadkowe usunięcie rurki), zatkanie drenażu, odleżyna (powstała w OAIIT).
  5. Przekazywanie Opieki: Ze względu na złożoność terapii, dokumentacja musi być czytelna i stanowić podstawę raportu przy zmianie dyżuru (metoda SBAR).
