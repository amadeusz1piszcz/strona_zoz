Dane Administracyjne i Kontekst Zabiegu

Identyfikacja pacjenta i planowanej procedury jako punkt wyjścia do oceny.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Identyfikacja Podmiotu

Pełna nazwa szpitala, kod resortowy, oddział kierujący na zabieg.

System / Admin

2. Identyfikacja Pacjenta

Imię i nazwisko, PESEL, Wiek (czynnik ryzyka), Płeć.

Pielęgniarka

3. Planowana Procedura

Nazwa zabiegu oraz przewidywany czas trwania (istotne dla ryzyka zakrzepowego i infekcyjnego).

Operator

4. Tryb Zabiegu

Planowy / Pilny / Nagły (tryb nagły automatycznie podnosi kategorię ryzyka).

Operator

Ocena Ryzyka Zatorowo-Zakrzepowego (VTE)

Skalowanie ryzyka żylnej choroby zakrzepowo-zatorowej (standard CMJ).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Skala Capriniego

(Zalecana dla chirurgii ogólnej/urologii) Suma punktów za czynniki: Wiek, Otyłość, Unieruchomienie, Nowotwór, Żylaki, Przebyta zakrzepica, Czas zabiegu >45 min.

Lekarz / Piel.

2. Skala dla Ortopedii

(Dla zabiegów ortopedycznych) Specyficzna ocena ryzyka przy endoprotezoplastyce stawów (THR/TKR) i złamaniach biodra.

Ortopeda

3. Poziom Ryzyka

Interpretacja wyniku: Ryzyko Niskie / Umiarkowane / Wysokie / Bardzo Wysokie.

System / Lekarz

4. Plan Profilaktyki

Wdrożone działania wynikające z oceny: Profilaktyka mechaniczna (pończochy, pompy), Farmakologiczna (LMWH - dawka, start), Wczesne uruchomienie.

Lekarz (zlec.)

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Klasa Czystości Rany

Przypisanie klasy rany (wg CDC): Czysta / Czysta-Skażona / Skażona / Brudna-Zakażona.

Operator

2. Skala ASA

Przeniesienie wyniku ze skali ASA (stan fizyczny pacjenta >2 zwiększa ryzyko zakażenia).

Anestezjolog

3. Czas Trwania (T)

Przewidywany czas trwania zabiegu przekraczający standardowy czas dla danej procedury (75. centyl).

Operator

4. Ryzyko Osobnicze

Cukrzyca (poziom HbA1c), Otyłość (BMI >30), Palenie tytoniu, Sterydoterapia, Immunosupresja.

Operator / Anest.

5. Nosicielstwo

Status MRSA/MSSA (jeśli wykonywano badania przesiewowe).

Piel. Epidemiol.

Ocena Ryzyka Żywieniowego

Wpływ stanu odżywienia na gojenie i powikłania.

Wymagana Zawartość Dokumentacyjna

1. Skala NRS 2002

(Lub SGA) Wynik oceny ryzyka niedożywienia. Wynik ≥3 pkt wymaga interwencji żywieniowej przed zabiegiem (jeśli możliwe).

Piel. / Dietetyk

2. Poziom Albumin

(Opcjonalnie) Stężenie albumin/białka całkowitego jako marker predykcyjny nieszczelności zespoleń i obrzęków.

Lekarz

Podsumowanie i Decyzje (Kwalifikacja Końcowa)

Synteza ryzyk i plan mitygacji.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Decyzja o Profilaktyce

Potwierdzenie zlecenia antybiotykoterapii okołooperacyjnej (Tak/Nie) adekwatnej do ryzyka.

Operator / Anest.

2. Przygotowanie Krwi

Decyzja o zabezpieczeniu preparatów krwiopochodnych (Krzyżówka/Na ratunek) w oparciu o ryzyko krwawienia.

Operator / Anest.

3. Zgoda na Ryzyko

Potwierdzenie, że zidentyfikowane ryzyka (np. bardzo wysokie ryzyko kardiologiczne przy zabiegu ratującym życie) zostały omówione z pacjentem w "Świadomej Zgodzie".

Operator

II. Standardy Akredytacyjne CMJ 2025 (Kluczowe Mechanizmy)

Karta Oceny Ryzyka jest dowodem na to, że szpital działa proaktywnie, a nie tylko reaktywnie.

  1. Profilaktyka Żylnej Choroby Zakrzepowo-Zatorowej (VTE):
    • Jest to jeden z najważniejszych standardów bezpieczeństwa. Szpital musi udowodnić, że każdy pacjent zabiegowy został oceniony pod kątem ryzyka zakrzepicy (z użyciem zwalidowanej skali, np. Caprini) i otrzymał profilaktykę adekwatną do wyniku. Brak takiej oceny w dokumentacji jest poważną niezgodnością.
  2. Monitorowanie Zakażeń (SSI):
    • Określenie "Klasy czystości rany" przed zabiegiem jest kluczowe dla późniejszej analizy zakażeń przez Zespół Kontroli Zakażeń Szpitalnych. Pozwala to na obliczenie współczynników zakażeń skorygowanych o ryzyko (benchmarking).
  3. Stan Odżywienia a Wyniki Leczenia:
    • Standardy wymagają identyfikacji pacjentów niedożywionych, ponieważ niedożywienie drastycznie zwiększa ryzyko rozejścia się zespoleń, zakażeń i zgonu. Dokumentacja musi wiązać wynik NRS ≥3 z wdrożeniem leczenia żywieniowego.
  4. Przygotowanie do Zabiegu:

Ocena ryzyka pozwala na bezpieczniejsze planowanie (np. zamówienie miejsca na OIT po zabiegu dla pacjenta z wysokim ASA i wieloma obciążeniami). Dokumentacja powinna odzwierciedlać ten proces decyzyjny.
