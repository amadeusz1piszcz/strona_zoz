KARTA INFORMACYJNA Z LECZENIA SZPITALNEGO

Jest to najważniejszy dokument medyczno-prawny podsumowujący hospitalizację. Zgodnie z prawem, musi stanowić syntetyczny, ale kompletny opis przebiegu leczenia, służący zapewnieniu ciągłości opieki w systemie ambulatoryjnym (POZ, AOS).

I. Specyfikacja Wymagań Dokumentacyjnych

Dane Administracyjne i Ruch Chorych

Formalne ramy pobytu określające czas i miejsce udzielania świadczeń.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Podmiot Leczniczy

Pełna nazwa szpitala, kod resortowy, nazwa oddziału wypisującego (oraz oddziałów, na których pacjent przebywał w trakcie pobytu - transfery).

System / Admin

2. Dane Pacjenta

Imię, nazwisko, PESEL, Adres zamieszkania (zgodny z deklaracją).

Sekretariat / System

3. Czas Pobytu

Data i godzina przyjęcia oraz Data i godzina wypisu. Liczba dni hospitalizacji.

System / Lekarz

4. Tryb Wypisu

Podstawa prawna: Zakończenie leczenia / Wypis na żądanie / Skierowanie do innego szpitala / Zgon / Wypis dyscyplinarny.

Lekarz

5. Orzeczenia

Informacja o wydaniu zwolnienia lekarskiego (e-ZLA): okres trwania, seria/numer (dla celów ZUS). Informacja o wydaniu wniosków (np. do sanatorium).

Lekarz

Rozpoznania i Procedury (Kodyfikacja)

Medyczny "bilans" pobytu w języku kodów ICD.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Rozpoznanie Główne

Ostateczna diagnoza będąca przyczyną hospitalizacji + Kod ICD-10 (musi być precyzyjny, np. I21.0 a nie tylko I21).

Lekarz Prowadzący

2. Rozpoznania Współistniejące

Lista chorób przewlekłych i problemów zdrowotnych leczonych lub monitorowanych podczas pobytu + Kody ICD-10.

Lekarz Prowadzący

3. Procedury Medyczne

Wykaz wykonanych zabiegów operacyjnych i istotnych procedur diagnostycznych (np. TK, Endoskopia) + Kody ICD-9 PL.

Lekarz / Koder

4. Przyczyna Zewnętrzna

(W przypadku urazów/zatruć) Kod przyczyny zewnętrznej (V, W, X, Y wg ICD-10) - niezbędne dla statystyki i NFZ.

Lekarz

Epikryza i Wyniki Badań 

Opisowa część historii choroby.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Zastosowane Leczenie

Opis wdrożonej terapii: Farmakoterapia wewnątrzszpitalna (grupy leków, antybiotyki), Zabiegi operacyjne (data, nazwa zabiegu), Rehabilitacja.

Lekarz Prowadzący

2. Przebieg Choroby (Epikryza)

Zwięzły opis ewolucji stanu pacjenta od przyjęcia do wypisu. Powikłania, reakcja na leczenie, uzyskana poprawa (lub jej brak).

Lekarz Prowadzący

3. Badania Laboratoryjne

Wybrane, istotne klinicznie wyniki badań (nie wszystkie!). Konieczne uwzględnienie wyników patologicznych oraz grupy krwi (jeśli oznaczano).

Lekarz (wybór)

4. Badania Obrazowe/Inne

Opisy badań: RTG, TK, MRI, USG, EKG, Endoskopia, Spirometria. W przypadku zabiegów - wynik badania Histopatologicznego (lub informacja o konieczności jego odbioru).

Lekarz

5. Konsultacje

Wnioski z odbytych konsultacji specjalistycznych (np. "Konsultacja kardiologiczna: wykluczono OZW").

Lekarz

Zalecenia Lekarskie i Farmakoterapia Domowa

Instrukcja dla pacjenta i lekarza POZ na przyszłość.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Leki do Domu (Recepty)

Lista leków zaleconych do stosowania przewlekłego. Nazwa, Dawka, Sposób dawkowania (np. 1-0-1). Wyraźne zaznaczenie zmian (np. "Odstawiono lek X", "Włączono nowy lek Y").

Lekarz Prowadzący

2. Kody Recept

Kody dostępowe do e-recept wystawionych przy wypisie (umożliwienie wykupienia leków).

System / Lekarz

3. Dalsze Leczenie

Wskazanie miejsca kontroli: Poradnia Specjalistyczna (AOS) / Lekarz Rodzinny (POZ). Termin kontroli (np. "za 14 dni").

Lekarz Prowadzący

4. Wskazania Dietetyczne

Ogólne zalecenia medyczne dot. diety (np. "Dieta łatwostrawna", "Ograniczenie soli"). Szczegóły znajdują się w zaleceniach pielęgniarskich/dietetycznych.

Lekarz Prowadzący

5. Autoryzacja

Podpis i pieczątka Lekarza Prowadzącego oraz Podpis i pieczątka Ordynatora/Kierownika Oddziału (wymóg prawny).

Lekarz + Ordynator

II. Zadania i Odpowiedzialność Poszczególnych Zawodów

  • Lekarz Prowadzący:
    • Synteza całego pobytu: wybór kluczowych badań do wpisania (nie należy "kopiować-wklejać" całego laboratorium, lecz wybrać wyniki istotne dla dalszego leczenia).
    • Precyzyjne ustalenie rozpoznania (ICD-10) i procedur (ICD-9).
    • Sformułowanie zaleceń farmakologicznych w sposób jasny dla pacjenta i lekarza POZ (rozróżnienie leków stałych od czasowych, np. przeciwbólowych).
    • Sporządzenie epikryzy (opisu przebiegu choroby).
  • Ordynator / Kierownik Oddziału:
    • Nadzór merytoryczny nad treścią karty.
    • Weryfikacja zgodności rozpoznania z przeprowadzonym procesem diagnostyczno-leczniczym.
    • Kontrasygnata (drugi podpis) na dokumencie.
  • Sekretarka Medyczna:
    • Wstępne przygotowanie danych administracyjnych (adres, PESEL).
    • Techniczne wprowadzanie danych pod dyktando lub nadzorem lekarza.
    • Wydanie dokumentu pacjentowi (za potwierdzeniem odbioru).

III. Standardy Akredytacyjne CMJ 2025 (Kluczowe Mechanizmy)

Karta Informacyjna jest wizytówką jakości szpitala w oczach lekarzy POZ i innych specjalistów.

  1. Ciągłość Opieki (Communication of Care):
    • Najważniejszy cel karty. Standardy wymagają, aby karta zawierała informacje wystarczające dla lekarza POZ do bezpiecznego przejęcia pacjenta. Musi być jasne: co zrobiono, co pacjent ma brać i co wymaga kontroli. Niejasne zalecenia (np. brak dawkowania leków) są poważną niezgodnością.
  2. Epikryza (Uzasadnienie):
    • Karta nie może być tylko zbiorem wyników badań. Musi zawierać "Epikryzę" – czyli wnioskowanie lekarskie. Dlaczego włączono ten lek? Dlaczego zmieniono diagnozę? Brak opisu przebiegu choroby obniża ocenę dokumentacji.
  3. Wyniki Badań (Histopatologia):
    • W przypadku zabiegów operacyjnych, standardem jest dołączenie wyniku badania histopatologicznego do karty. Jeśli wynik jest niedostępny w dniu wypisu, karta musi zawierać wyraźną informację: "Wynik hist-pat do odbioru w terminie X. Konieczna kontrola z wynikiem". Brak tej informacji stwarza ryzyko przeoczenia choroby nowotworowej.
  4. Przejrzyste Zalecenia Lekowe:
    • W sekcji zaleceń musi być jasność, które leki przyjmowane przez pacjenta przed szpitalem są kontynuowane, a które odstawione. Zapis "Leki wg karty wypisowej" jest niewystarczający, jeśli nie ma odniesienia do leków "domowych".
  5. Kompletność Podpisów:
    • Wymóg prawny i akredytacyjny: Karta musi być podpisana przez lekarza prowadzącego ORAZ kierownika oddziału (lub jego zastępcę). Brak dwóch podpisów czyni dokument wadliwym prawnie.
