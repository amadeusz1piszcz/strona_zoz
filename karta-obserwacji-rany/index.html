KARTA OBSERWACJI RANY

I. Specyfikacja Wymagań Dokumentacyjnych

Dane Administracyjne i Identyfikacja

Dane statyczne przypisujące dokument do pacjenta, określające ryzyko i etiologię.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Identyfikacja Podmiotu

Pełna nazwa podmiotu leczniczego, kod resortowy, nazwa oddziału.

Administrator Systemu, Rejestracja

2. Identyfikacja Pacjenta

Imię i nazwisko, PESEL (lub data ur.), numer Księgi Głównej.

Rejestracja / Piel.

3. Ryzyko Powstania Ran

Wynik w skali ryzyka odleżyn (np. Skala Norton, Braden, Douglas) – data i wynik punktowy.

Pielęgniarka

4. Choroby Współistniejące

Czynniki utrudniające gojenie: Cukrzyca, Miażdżyca, Niedożywienie, Immunosupresja, Otyłość.

Lekarz / Piel.

Inicjacja (Charakterystyka Wstępna Rany)

Pierwsza ocena po przyjęciu pacjenta lub po wystąpieniu rany w trakcie hospitalizacji.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Czas Powstania/Zgłoszenia

Data (DD-MM-RRRR) ujawnienia rany lub przyjęcia z raną. Określenie: "Rana zastana" vs "Rana nabyta w szpitalu".

Pielęgniarka

2. Etiologia (Rodzaj)

Typ: Odleżyna (stopień wg NPUAP/EPUAP), Owrzodzenie żylne/tętnicze, Stopa cukrzycowa, Rana chirurgiczna, Oparzenie.

Lekarz / Piel.

3. Lokalizacja

Dokładny opis anatomiczny (np. "Okolica kości krzyżowej", "Pięta lewa") lub oznaczenie na diagramie ciała (mapa ciała).

Pielęgniarka

4. Wymiary Wyjściowe

Długość (cm), Szerokość (cm), Głębokość (cm) lub powierzchnia (cm²).

Pielęgniarka

5. Dokumentacja FOTO

(Opcjonalne, Zalecane) Informacja o wykonaniu zdjęcia rany do dokumentacji (wymagana zgoda pacjenta > formularz).

Pielęgniarka

Monitorowanie (Ocena Procesu Gojenia)

Wypełniana przy każdej zmianie opatrunku (zgodnie ze strategią TIME).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Czas Oceny

Data i godzina zmiany opatrunku/oceny rany.

Pielęgniarka

2. Tkanka (T - Tissue)

Opis dna rany: Martwica czarna (nekroza), Martwica rozpływna (włóknik żółty), Ziarnina (czerwona), Naskórkowanie (różowa). % udziału tkanek.

Pielęgniarka

3. Infekcja/Zapalenie (Infection)

Objawy: Zaczerwienienie, obrzęk, ocieplenie, ból, nieprzyjemny zapach.

Pielęgniarka

4. Wilgotność (M - Moisture)

Ilość wysięku: Brak, Mały, Średni, Duży. Rodzaj: Surowiczy, Ropny, Krwisty.

Pielęgniarka

5. Brzegi Rany (E - Edge)

Stan brzegów: Prawidłowe, Zmacerowane, Podminowane, Wałowate. Stan skóry wokół rany.

Pielęgniarka

6. Pomiar Rany

Aktualne wymiary (długość x szerokość x głębokość). Ocena obecności przetok/kieszeni.

Pielęgniarka

7. Poziom Bólu

Ocena bólu proceduralnego (przy zmianie opatrunku) i ciągłego wg skali NRS (0-10).

Pielęgniarka

Pielęgnacja (Leczenie Miejscowe)

Rejestr wykonanych procedur medycznych na ranie.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Oczyszczanie (Mycie)

Użyty preparat: (np. 0.9% NaCl, płyn z poliheksanidyną, oktenidyna, podchloryny). Metoda: Przymoczki / Płukanie / Irygacja.

Pielęgniarka

2. Opracowanie

Metoda usunięcia martwicy: Chirurgiczna (skalpel), Mechaniczna, Autolityczna (żele), Enzymatyczna.

Lekarz / Piel.

3. Opatrunek Pierwotny

Rodzaj opatrunku stykającego się z dnem rany (np. hydrożel, alginian, srebro, pianka).

Pielęgniarka

4. Opatrunek Wtórny

Rodzaj opatrunku zabezpieczającego (np. kompres gazowy, plaster, bandaż).

Pielęgniarka

5. Zalecenia Dodatkowe

Zastosowane odciążenie (np. materac zmiennociśnieniowy, krążek), ułożenie kończyny.

Pielęgniarka

Zakończenie (Wynik Leczenia)

Podsumowanie terapii dla danej rany.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Data Zakończenia

Data ostatniej obserwacji (wypis/zgon/wygojenie).

Pielęgniarka

2. Stan Końcowy

Wybór: Wygojenie całkowite, Poprawa (zmniejszenie powierzchni), Brak poprawy/Pogorszenie, Zgon pacjenta.

Piel. / Lekarz

3. Zalecenia przy Wypisie

Informacja o przekazaniu pacjentowi/rodzinie instrukcji dalszej pielęgnacji rany w domu (wymóg edukacji zdrowotnej).

Pielęgniarka

II. Standardy Akredytacyjne CMJ 2025 (Kluczowe Mechanizmy)

Dokumentacja rany jest dowodem na jakość opieki i musi realizować następujące standardy:

  1. Strategia TIME: Standardy promują ujednolicony język opisu rany. Karta powinna wymuszać ocenę 4 kluczowych elementów: Tkanki (Tissue), Infekcji (Infection), Wilgoci (Moisture), Brzegów (Edge). Opis typu "rana czysta" jest niewystarczający.
  2. Monitorowanie Zakażeń (SSI): W przypadku ran pooperacyjnych, karta jest podstawowym źródłem informacji dla Zespołu Kontroli Zakażeń o wystąpieniu Zakażenia Miejsca Operowanego (ZMO).
  3. Profilaktyka Odleżyn: Dokumentacja musi wykazywać korelację między oceną ryzyka (Skala Braden/Norton w Metryce) a podjętymi działaniami (wpisy o materacach i zmianie pozycji w sekcji Pielęgnacja).
  4. Mierzalność Postępów: Wymóg regularnego mierzenia powierzchni rany (długość x szerokość). Brak postępów w gojeniu w określonym czasie powinien skutkować odnotowaną weryfikacją metody leczenia.
