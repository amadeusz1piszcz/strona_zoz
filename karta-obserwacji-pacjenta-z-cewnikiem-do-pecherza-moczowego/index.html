KARTA OBSERWACJI PACJENTA Z CEWNIKIEM DO PĘCHERZA MOCZOWEGO

I. Specyfikacja Wymagań Dokumentacyjnych

Dane Administracyjne i Identyfikacja

Dane statyczne przypisujące dokument do pacjenta i epizodu leczenia.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Identyfikacja Podmiotu

Pełna nazwa podmiotu leczniczego, kod resortowy (część V i VII), nazwa komórki organizacyjnej (oddziału).

Administrator Systemu, Rejestracja

2. Identyfikacja Pacjenta

Imię i nazwisko, PESEL (lub data urodzenia i płeć), numer Księgi Głównej (historia choroby).

Rejestracja / Piel.

3. Lokalizacja Pacjenta

Numer sali chorych, numer łóżka (niezbędne dla organizacji pracy i bezpieczeństwa).

Pielęgniarka

4. Wskazania i Ryzyka

Powód cewnikowania (np. "Zatrzymanie moczu", "Monitorowanie diurezy godzinowej", "Zabieg urologiczny") oraz alerty (np. "Przerost prostaty").

Lekarz

Inicjacja (Procedura Założenia Cewnika)

Dokumentacja procedury inwazyjnej (rozpoczęcie ekspozycji na ryzyko ZUM).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Czas Założenia

Data (DD-MM-RRRR) i godzina (GG:MM) wykonania procedury.

Osoba zakładająca

2. Rodzaj Cewnika

Typ (np. Foley’a lateksowy, silikonowy 100%, Tiemann), Rozmiar (Ch/Fr), Pojemność balonu (ml).

Osoba zakładająca

3. Przebieg Procedury

Potwierdzenie aseptyki (dezynfekcja ujścia cewki), ilość wody w balonie uszczelniającym (ml), opis trudności (np. "opór w części sterczowej").

Osoba zakładająca

4. Wypływ Moczu

Potwierdzenie uzyskania wypływu moczu po założeniu (Tak/Nie) oraz jego wygląd (jasny/krwisty/mętny).

Osoba zakładająca

5. Autoryzacja

Podpis i pieczątka (lub ID elektroniczne) osoby wykonującej zabieg.

Osoba zakładająca

Monitorowanie (Obserwacja Bieżąca)

Wypełniana cyklicznie (min. raz na zmianę pielęgniarską).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Czas Oceny

Data i godzina bieżącego monitorowania.

Pielęgniarka

2. Bilans Płynów

Ilość wydalonego moczu (ml) w zadanym okresie (bilans dobowy/zmianowy).

Pielęgniarka

3. Ocena Moczu

Charakterystyka makroskopowa: Barwa, Klarowność (klarowny/mętny/z osadem), Obecność krwi (krwiomocz).

Pielęgniarka

4. Drożność Systemu

Potwierdzenie drożności drenów, brak załamań, swobodny spływ (grawitacyjny).

Pielęgniarka

5. Ocena Zasadności

Kluczowy wymóg CMJ: Decyzja "Czy utrzymanie cewnika jest nadal niezbędne?" (TAK/NIE).

Lekarz / Piel.

6. Komfort Pacjenta

Zgłaszane dolegliwości: ból, pieczenie, parcie na pęcherz (objawy mogące sugerować zakażenie lub niedrożność).

Pielęgniarka

Pielęgnacja

Rejestr czynności profilaktycznych przeciw zakażeniom układu moczowego (CAUTI).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Toaleta Krocza

Potwierdzenie wykonania toalety okolicy ujścia cewki moczowej (woda z mydłem / środek antyseptyczny błon śluzowych).

Pielęgniarka

2. Pozycja Worka

Potwierdzenie, że worek znajduje się poniżej poziomu pęcherza moczowego (zapobieganie refluksowi moczu) i nie dotyka podłogi.

Pielęgniarka

3. Integralność Układu

Potwierdzenie, że układ jest zamknięty (nie był rozłączany). Jeśli rozłączono – przyczyna i data wymiany worka.

Pielęgniarka

4. Wymiana Worka

Data wymiany worka na mocz (zgodnie z procedurą szpitalną lub w razie uszkodzenia/nieszczelności).

Pielęgniarka

Zakończenie (Procedura Usunięcia)

Zamknięcie cyklu życia dostępu do pęcherza.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Czas Usunięcia

Data i godzina usunięcia cewnika (koniec inwazyjnej ekspozycji).

Pielęgniarka

2. Przyczyna

Wskazanie powodu: Powrót spontanicznej mikcji, Podejrzenie zakażenia (CAUTI), Niedrożność/Zatkanie, Planowa wymiana.

Piel. / Lekarz

3. Integralność

Potwierdzenie usunięcia cewnika w całości (czy balon nie uległ fragmentacji).

Pielęgniarka

4. Mikcja po Usunięciu

Obserwacja: Czy pacjent oddał mocz samodzielnie po usunięciu cewnika? (TAK/NIE, czas pierwszego oddania moczu).

Pielęgniarka

5. Stan Pacjenta

Ocena pod kątem retencji moczu po usunięciu (np. badanie palpacyjne pęcherza lub USG pęcherza).

Lekarz / Piel.

 

II. Standardy Akredytacyjne CMJ 2025 (Kluczowe Mechanizmy)

Wymagania, które muszą być odzwierciedlone w strukturze formularza w kontekście prewencji odcewnikowych zakażeń układu moczowego (CAUTI):

  1. Weryfikacja Wskazań: Szpital musi udowodnić, że codziennie oceniana jest zasadność utrzymania cewnika. Przedłużające się cewnikowanie bez jasnych wskazań jest główną przyczyną zakażeń szpitalnych.
  2. Zamknięty System Drenażu: Dokumentacja musi potwierdzać utrzymanie ciągłości układu (cewnik-dren-worek). Rozłączanie układu powinno być zdarzeniem wyjątkowym i odnotowanym.
  3. Higiena i Pozycjonowanie: Formularz powinien wymuszać kontrolę pozycji worka (zawsze poniżej pęcherza) jako kluczowego elementu zapobiegania wstecznemu odpływowi moczu.

Monitorowanie Diurezy: Precyzyjne bilansowanie płynów jest nie tylko elementem terapii, ale też wczesnym wskaźnikiem powikłań (np. skąpomocz, niedrożność cewnika)
