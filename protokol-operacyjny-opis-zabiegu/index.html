PROTOKÓŁ OPERACYJNY / OPIS ZABIEGU

I. Specyfikacja Wymagań Dokumentacyjnych

Dane Administracyjne i Zespół Operacyjny

Dane identyfikacyjne, czasowe i personalne definiujące odpowiedzialność za wykonany zabieg.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Identyfikacja Podmiotu

Pełna nazwa szpitala, kod resortowy, nazwa oddziału zlecającego oraz nazwa bloku operacyjnego.

System / Admin

2. Identyfikacja Pacjenta

Imię i nazwisko, PESEL, numer Księgi Głównej, Wiek.

Pielęgniarka

3. Ramy Czasowe

Data zabiegu, Godzina rozpoczęcia (nacięcia skóry) i Godzina zakończenia (założenia opatrunku). Czas trwania zabiegu.

Piel. / Operator

4. Tryb Zabiegu

Określenie trybu: Planowy / Pilny / Nagły (ratujący życie).

Operator

5. Zespół Chirurgiczny

Imię i nazwisko Operatora (główny chirurg), Asysty I, Asysty II (lekarze).

Operator

6. Zespół Wspierający

Imię i nazwisko Pielęgniarki Instrumentariuszki, Pielęgniarki Pomocniczej oraz Lekarza Anestezjologa (dla celów koordynacji).

Pielęgniarka

Rozpoznanie i Procedura (Dane Kliniczne)

Kodyfikacja medyczna wykonanych czynności.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Rozpoznanie Przedoperacyjne

Diagnoza będąca podstawą kwalifikacji do zabiegu (wraz z kodem ICD-10).

Operator

2. Rozpoznanie Pooperacyjne

Ostateczna diagnoza ustalona po otwarciu pola operacyjnego/wykonaniu zabiegu (wraz z kodem ICD-10).

Operator

3. Nazwa Zabiegu

Pełna nazwa słowna wykonanej procedury (np. "Laparoskopowe wycięcie pęcherzyka żółciowego").

Operator

4. Kodyfikacja Procedur

Kod procedury wg klasyfikacji ICD-9 PL (wersja aktualna). Możliwość wpisania wielu procedur (główna + towarzyszące).

Operator / Koder

5. Strona Operowana

Wyraźne wskazanie strony ciała (Prawa / Lewa / Obustronnie) - dotyczy narządów parzystych.

Operator

Opis Przebiegu Operacji (Narracja)

Szczegółowe sprawozdanie z techniki operacyjnej (od nacięcia do zamknięcia).

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Przygotowanie Pola

Opis ułożenia pacjenta, sposób mycia i obłożenia pola operacyjnego (antyseptyka).

Operator

2. Dostęp Operacyjny

Opis rodzaju i miejsca nacięcia skóry (np. "Cięcie pośrodkowe", "Trokar w pępku").

Operator

3. Eksploracja i Obraz

Opis warunków anatomicznych zastanych w polu operacyjnym (np. "Zrosty", "Wodniak", "Guz naciekający").

Operator

4. Technika Operacyjna

"Krok po kroku" opis wykonanych czynności naprawczych, resekcyjnych lub rekonstrukcyjnych. Użyte techniki (np. "Szew ciągły", "Zespolenie mechaniczne").

Operator

5. Hemostaza

Sposób zaopatrzenia krwawienia (np. "Koagulacja dwubiegunowa", "Podkłucie naczynia").

Operator

6. Zamknięcie

Sposób zamknięcia powłok (rodzaj szwów dla poszczególnych warstw: powięź, tkanka podskórna, skóra). Rodzaj opatrunku.

Operator

Bezpieczeństwo i Materiały

Elementy kluczowe dla identyfikowalności użytych materiałów i bezpieczeństwa.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Wszczepy (Implanty)

Rejestr użytych implantów (np. siatki przepuklinowe, endoprotezy, śruby, stenty). Wklejenie "paszportu implantu" (kod EAN, numer serii/lot, data ważności).

Piel. / Operator

2. Materiał Biologiczny

Spis pobranych wycinków do badania histopatologicznego lub bakteriologicznego. Oznaczenie pojemników.

Operator

3. Drenaż

Informacja o pozostawionych drenach (rodzaj, lokalizacja, czy podłączono do ssania).

Operator

4. Kontrola Zgodności

Potwierdzenie zgodności materiału i narzędzi (Gazy/Narzędzia) przed zamknięciem powłok. Wpis: "Zgodne / Niezgodne".

Piel. Instrument.

5. Utrata Krwi

Szacunkowa objętość utraconej krwi (ml).

Operator / Anest.

Powikłania i Zalecenia

Zdarzenia niepożądane i instrukcje dla oddziału pooperacyjnego.

Element Zlecenia/Realizacji

Wymagana Zawartość Dokumentacyjna

Odpowiedzialność

1. Powikłania Śródoperacyjne

Opis zdarzeń niepożądanych: Uszkodzenie sąsiednich narządów, Krwotok, Awarie sprzętu, Konwersja (zamiana laparoskopii na zabieg otwarty).

Operator

Wskazówki dla oddziału: Ułożenie pacjenta, Antybiotykoterapia kontynuacyjna, Profilaktyka przeciwzakrzepowa, Pielęgnacja rany/drenu.

Operator

II. Standardy Akredytacyjne CMJ 2025 (Kluczowe Mechanizmy)

Protokół operacyjny jest fundamentalnym dokumentem prawnym i jakościowym. Wymagania akredytacyjne obejmują:

  1. Niezwłoczność Sporządzenia: Standardy wymagają, aby opis operacji powstał bezpośrednio po zabiegu, zanim operator przystąpi do kolejnych czynności (lub opuści szpital). Opóźnienia w opisywaniu zabiegów (np. "pod koniec dnia") stwarzają ryzyko błędów pamięciowych i są niezgodnością.
  2. Identyfikowalność Implantów: W przypadku użycia wszczepów, dokumentacja musi pozwalać na pełną identyfikację elementu (producent, typ, rozmiar, numer serii). Jest to krytyczne w przypadku ogłoszenia alertów bezpieczeństwa przez producenta (konieczność wezwania pacjenta).
  3. Weryfikacja Rozpoznania: Protokół musi wykazywać porównanie rozpoznania przedoperacyjnego z pooperacyjnym. Rozbieżności powinny być omówione/wyjaśnione (jest to element oceny jakości diagnostyki przedoperacyjnej).
  4. Raportowanie Zdarzeń Niepożądanych: Wszelkie uszkodzenia jatrogenne (przypadkowe naruszenie struktur, np. jelita, moczowodu) muszą być odnotowane w protokole w sposób jawny i transparentny, wraz z opisem sposobu ich zaopatrzenia.
  5. Czytelność i Kompletność: Opis "Zabieg wykonano typowo" jest niedopuszczalny w świetle standardów jakości. Opis musi być szczegółowy i odwzorowywać faktyczny przebieg, w tym napotkane trudności techniczne.
  6. Kontrola Materiału: Adnotacja o zgodności materiału gazowego i narzędzi przed zamknięciem jam ciała jest bezwzględnym wymogiem bezpieczeństwa (zapobieganie pozostawieniu ciał obcych).
